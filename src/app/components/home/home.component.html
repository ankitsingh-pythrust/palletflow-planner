What Is PalletFlow Planner?

Built for warehouses, 3PLs, and dispatch teams — a web app that streamlines load building from intake to dispatch without spreadsheets or double entry.

  • Design shipments with a mixed list of pallets and loose items
  • Auto‑surface pallet details only when relevant

Cloud-hosted with regional data residency options.

How It Works

From intake to dispatch in five quick steps — scan, expand, drag, validate, export.

1) Build your source list
Add items manually, import CSV, or scan barcodes. Flag items as pallet or non‑pallet.
2) Auto‑expand pallets
View dependent details (SKUs, counts, weight, dimensions) on demand.
3) Create placeholders
Empty slots for trucks/containers with capacity and rules.
4) Drag to assign
Drag items into placeholders. See live capacity checks.
5) Validate & export
Lock loads, generate BOL/packing lists, and export to TMS/CSV or via API.

Answers to common questions

If you don't see yours, we're one message away.

Can I keep non‑pallet items simple?

Yes — enter them as single lines without dependent details. Pallet items expand only when needed.

 How do placeholders work?

Create named slots for each truck/container and stop, with capacity and rules. Drag items into these slots and validate instantly.

 What capacity rules are supported?

Weight, cube, axle, stacking, hazmat, temperature, and segregation — configured per profile.

 Do you support scanners?

Yes — Zebra and Honeywell, plus camera scanning on mobile browsers.

 Is there an API?

Yes — REST API and SFTP/CSV for batch jobs. Webhooks for real‑time events.

 What about deployment?

Cloud by default; private cloud or on‑prem options for Enterprise customers.
